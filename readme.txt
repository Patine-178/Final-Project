วิธี run project
1. คลิกขวาที่ folder "Final-Project"
2. เลือก "Open with code" (VS Code) 
3. คลิกขวาที่ folder "flask-project" ใน VS Code
4. เลือก "Open in Integrated Terminal"
5. พิมพ์คำสั่งใน terminal -> flask run
6. เปิด terminal มาอีก 1 หน้าเพื่อ run client
7. ไปยัง path "Final-Project/client"
8. พิมพ์คำสั่งใน terminal -> python app.py

URL
http://127.0.0.1:5000/ คือ API model
http://127.0.0.1:5000/key คือ หน้า web page ที่จะขอ API-Key
http://127.0.0.1:5000/allKey คือ API-Key ทั้งหมดที่สร้างมา
http://127.0.0.1:8000/client/home คือ หน้า web page หลักของ client
